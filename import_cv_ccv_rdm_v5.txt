Voici la version robuste en Python.
Elle fait tout en un seul script :
crée les CV simples avec le même name et le même label
ajoute les repositories aux CV en recherchant les repositories par name
crée les CCV avec le même name et le même label
ajoute les components CV aux CCV
génère un rapport CSV de succès / erreurs
Elle est plus fiable que Bash parce qu’elle lit correctement les CSV, même si un champ contient des virgules.
Exécution
Bash
Copy code
chmod +x import_all_cv_ccv.py
./import_all_cv_ccv.py
ou :
Bash
Copy code
python3 import_all_cv_ccv.py
Ce que le script conserve
Pour chaque CV et CCV :
même nom
même label
Pour les repositories :
recherche par repository name
pas par label
Pour les components de CCV :
recherche par nom du Content View composant
Rapport généré
Le script produit un fichier du type :
Bash
Copy code
import_report_20260314_134500.csv
avec des statuts comme :
CREATED
ATTACHED
OK_EXISTS
CONFLICT
ERROR
