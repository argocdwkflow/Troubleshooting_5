Bonne idée 👍 — dans ton cas plusieurs PID SAS appartiennent au même utilisateur (pamow01), donc il faut calculer la consommation CPU totale par utilisateur.
Voici les méthodes utilisées par les admins Linux N3.
1️⃣ Calcul CPU total par utilisateur (simple)
Bash
Copy code
ps -eo user,%cpu --no-headers | awk '{cpu[$1]+=$2} END {for (u in cpu) printf "%s %.2f%%\n", u, cpu[u]}'
Exemple résultat :
Copy code

pamow01 320.5%
root 5.2%
pfass 2.1%
⚠️ 320% = 3.2 CPU utilisés.
2️⃣ Voir seulement SAS pour cet utilisateur
Bash
Copy code
ps -u pamow01 -o pid,%cpu,cmd --sort=-%cpu | grep sas
3️⃣ Calcul CPU total pour SAS uniquement
Bash
Copy code
ps -u pamow01 -o %cpu,cmd | grep sas | awk '{sum+=$1} END {print "Total CPU:",sum,"%"}'
Exemple :
Copy code

Total CPU: 285.4 %
Donc SAS utilise ≈ 3 CPU.
4️⃣ Voir combien de CPU SAS utilise réellement
Bash
Copy code
ps -u pamow01 -o pid,%cpu,cmd | grep sas
Puis :
Bash
Copy code
ps -u pamow01 | wc -l
