Voici le script d’export complet qui génère les 3 fichiers utilisés par l’import :
content_views_full.csv
content_view_repositories.csv
ccv_components.csv
Il exporte :
chaque Content View / Composite Content View
leur name
leur label
leur statut composite
leur description
les repositories de chaque CV avec repo id / repo name / repo label
les components de chaque CCV

Exécution
Bash
Copy code
chmod +x export_all_cv_ccv.py
./export_all_cv_ccv.py
ou :
Bash
Copy code
python3 export_all_cv_ccv.py
Fichiers générés

Bash
Copy code
