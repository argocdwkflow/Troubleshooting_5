Voici le script export en shell qui génère :
content_views_full.csv
content_view_repositories.csv
ccv_components.csv
Il exporte :
chaque Content View / Composite Content View
leur name
leur label
leur statut composite
leur description
les repositories de chaque CV avec repo id / repo name / repo label
les components de chaque CCV

Exécution
Bash
Copy code
chmod +x export_all_cv_ccv.sh
./export_all_cv_ccv.sh
Fichiers générés
content_views_full.csv
content_view_repositories.csv
ccv_components.csv
Exemple de structure
Bash
Copy code
sat_export_20260314_150000/
├── content_views_full.csv
├── content_view_repositories.csv
├── ccv_components.csv
├── export.log
└── raw_content_views/
